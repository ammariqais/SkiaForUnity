To use the {\ttfamily Skottie\+Player} component, follow these steps\+:


\begin{DoxyEnumerate}
\item {\bfseries{Create a Raw\+Image\+:}}

In your Unity scene, create a {\ttfamily Raw\+Image} object to display the Skottie animation. You can do this by right-\/clicking in the Hierarchy panel, selecting "{}\+UI,"{} and choosing "{}\+Raw Image."{}
\item {\bfseries{Attach Skottie\+Player Component\+:}}

Select the {\ttfamily Raw\+Image} you created. In the Inspector panel, click the "{}\+Add Component"{} button and search for "{}\+Skottie Player"{} to add the {\ttfamily Skottie\+Player} component to the {\ttfamily Raw\+Image} object.
\item {\bfseries{Assign Animation JSON Data\+:}}

In the Inspector panel, you\textquotesingle{}ll find the {\ttfamily Skottie\+Player} component you just added. Inside the {\ttfamily Skottie\+Player} component, you\textquotesingle{}ll see a field named "{}lottie\+File"{} This is where you can assign a Text\+Asset containing your animation JSON data. To do this, drag and drop your JSON file into the "{}\+Animation Json File"{} field.
